## Front-End Developer (React.js) — 3–4 Years

We’re looking for a **strong Front-End Developer** who enjoys building real products, solving real problems, and taking ownership end-to-end. This role is for someone who can **think independently, write logic without AI shortcuts**, and ship **pixel-perfect, production-ready UI**.

If you rely on reading docs, debugging deeply, and figuring things out instead of waiting for answers — you’ll fit right in.

---

### What You’ll Do

* Build and scale frontend features using **React.js**
* Convert **Figma designs into pixel-perfect, responsive UI**
* Use **Tailwind CSS** and **Ant Design (AntD)** to build clean interfaces
* Manage state using **Redux, Redux Persist**, and **Context API**
* Integrate APIs and handle async flows confidently
* Integrate **third-party SDKs** and external libraries
* Debug issues across environments — **including production**
* Track, investigate, and fix errors using **source maps, Sentry, or similar tools**
* Improve performance, stability, and overall user experience
* Work closely with designers, backend engineers, and product teams

---

### What We’re Looking For

* 3–4 years of hands-on frontend development experience
* Strong, real-world experience with **React.js**
* Solid command over:

  * Tailwind CSS
  * Ant Design
  * Redux & Redux Persist
  * Context API
* Strong JavaScript fundamentals (ES6+)
* Comfortable with API integration and real production data
* Ability to **write business logic independently (no AI dependency)**
* Strong habit of reading documentation and figuring things out
* Excellent debugging and troubleshooting skills
* Experience handling production issues calmly and logically
* Strong eye for detail and UI accuracy

---

### Bonus Points If You Have

* Experience with **Next.js**
* Knowledge of performance optimization
* Experience in fast-moving startup environments
